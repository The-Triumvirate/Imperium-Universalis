# SLAVERY PERCENTAGE NEW MODIFIERS
####
#
# Now the modifiers are done in a "table" of A) Slave population % -> Society effects, and B) Slave % economic output -> Economic effects
# Society effects will only become worse the bigger it goes, while economic effects are positive.
# The main thing is that you will not have good productivity, without social unrest.
#
#	SLAVE PRODUCTION %, Y axis:
#	1:0-10  2:10-20	 3: 20-30  4:30-40	 5:40-50  6:50-60  7:60-80 8:80-100
#	9:100-120  10:120-140  11:140-160  12:160-200  13:200-250  14:250-300  15:>300
#
# 	SLAVE POPULATION %, X axis:
#	1: 0-10%	2: 10-25%	3: 25-50%	4: 50-80%	5: 80-120%	6: 120-200%		7: >200%. You better not reach over 50%
#
#	This means we have 7 for population, and 15 for productivity. 7 x 15 = 105 potential modifiers. Let's go
