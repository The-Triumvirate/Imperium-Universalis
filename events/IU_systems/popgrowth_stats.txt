####
# POPGROWTH, STATISTICS EVENTS
####
# 1: Statistics menu: choose the following (through country decision)
# 	2: Pop variables into your country, to save and analyze separately in excel
#	3: How has your country evolved: population (u/c/t + var) by region, number of provinces (and change) also present
#		301: How has your country evolved. Including your vassals (western mediterranean, for Rome games)
#		302: How has your country evolved. Including your vassals (east med / persia, for Persia/Babylon games)
#		303: How has your country evolved. Including your vassals (far east, for China games)
# 	4: Display pop analysis of the selected provinces, depending on total scope:
#		Possible scopes: In prov, area, region, super-region, within 200, 500, 1000 distance, or owned (only, or with vassals), or all.
#		401: Show detailed info on pop capacities.
#		402: Main variations, together (as in the event 3, but for the selected area). This event chooses the scope, and is calculated in 403
#		403: the effect of the above
#	5: Icons in the map:
#				In the map modifiers, everything will have a color, from a 16-color gradient
#		500: Province size (1 to 10)
#		501: Rural: abs + growth
#		502: Urban: abs + growth
#		503: Upper: abs + growth
#		504: Potential: % left to grow (-20%, -10, 0, 10, 20, 30, up to 90%+)
#		505: Effective information range (migration range)
#		506: How many X things has there been, positive effects: grown + promotions + immigration
#		507: How many X things has there been, negative effects: famine + demotions + emigration
#		508: How many X things has there been, war-related effects: enslaved + killed
#
#	6: Province detailed evolution: more information related to a single province; accessible through province interaction. Province only
#
#	7: Popstats Dev Tools(tm), very powerful events that will affect all the world's population depending on current capacities.
#
